Student Plan Assistant (SPA) is a software engineering class project created by Ben Doan, Brock Lewis, Patrick Lynch, and James Percival. Our goal for this project was to create an easy-to-use application for students and advisors to create a viable future schedule for completing a degree. The user chooses the classes they have already completed and the application generates their remaining requirements and electives.


SPA is written in Python using the Flask web framework. Flask uses a SQLite implementation for its persistent storage. It uses Bootstrap to drive the look and feel of the site through CSS.
